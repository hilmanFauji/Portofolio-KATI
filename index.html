Skill

Kemampuan Saya di Beberapa Bidang

Selama masa-masa hidup saya, saya telah mempelajari berbagai bidang yang melatih skill dan kreatifitas saya.

HTML

HTML adalah bahasa markup yang digunakan untuk membuat halaman web dengan struktur dan tampilan konten yang interaktif.

Software Blender

Blender adalah perangkat lunak grafis 3D gratis dan sumber terbuka yang digunakan untuk membuat animasi dan model.

Filmora adalah perangkat lunak video yang intuitif dengan efek kreatif dan fitur pengeditan yang lengkap.

CSS

CSS atau Cascading Style Sheets adalah bahasa yang digunakan untuk mengatur tampilan dan gaya elemen dalam halaman web.

Bahasa C

Bahasa C adalah bahasa pemrograman tingkat menengah yang digunakan untuk pengembangan perangkat lunak dan sistem operasi.

Bahasa C++

C++ adalah bahasa pemrograman tingkat menengah yang mendukung paradigma pemrograman berorientasi objek.
